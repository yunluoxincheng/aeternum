# Aeternum 用户指南

**版本**: 1.0.0
**最后更新**: 2026-02-16

---

## 概述

Aeternum 是一款后量子安全的移动密钥管理应用，采用 ML-KEM (Kyber-1024) 加密技术保护您的数字资产安全。本指南将帮助您了解如何使用 Aeternum 的各项功能。

### 核心特性

- **后量子安全**: 采用 ML-KEM (Kyber-1024) + X25519 混合加密
- **多设备同步**: 支持多设备间的密钥同步与管理
- **48小时否决机制**: 防止助记词被盗后的未经授权恢复
- **生物识别保护**: 使用指纹或面容识别保护您的数据

---

## 快速开始

### 首次使用

1. **启动应用**: 打开 Aeternum 应用，您将看到欢迎屏幕
2. **备份助记词**: 系统会生成 24 位助记词，请务必将其抄写在安全的地方
3. **确认备份**: 按照屏幕提示确认您已安全保存助记词
4. **设置生物识别**: 设置指纹或面容识别作为日常解锁方式
5. **开始使用**: 完成设置后，您即可开始使用 Aeternum

### 助记词安全提示

助记词是您恢复账户的唯一方式，请务必：

- 将助记词抄写在纸上，存放在安全的地方
- 不要将助记词存储在电子设备或云端
- 不要向任何人透露您的助记词
- 定期检查助记词的保存状态

## 主要功能

### 1. 主屏幕

主屏幕显示当前安全状态和快速操作入口：

- **状态卡片**: 显示当前纪元、设备状态等信息
- **快速操作**: 快速访问常用功能
- **最近活动**: 显示最近的操作记录

### 2. 设备管理

在设备管理页面，您可以：

- **查看设备列表**: 查看所有已连接的设备
- **添加新设备**: 通过扫描 QR 码添加新设备
- **查看设备详情**: 查看设备的详细信息（纪元、状态、最后活跃时间等）
- **撤销设备**: 撤销不再使用的设备

#### 设备状态说明

| 状态 | 说明 |
|------|------|
| 活跃 | 设备正常工作，可以进行所有操作 |
| 降级 | 设备完整性验证失败，仅可查看数据 |
| 已撤销 | 设备已被撤销，无法访问数据 |

### 3. 恢复账户

如果您的设备丢失或需要在新设备上访问账户：

1. **打开恢复页面**: 从主屏幕进入恢复功能
2. **输入助记词**: 按顺序输入您的 24 位助记词
3. **等待确认**: 系统将进入 48 小时否决窗口
4. **完成恢复**: 如果没有其他设备否决，恢复将在 48 小时后完成

### 4. 否决机制

当有其他设备尝试恢复账户时：

- 您将收到恢复通知
- 在 48 小时内，您可以点击"否决"按钮阻止恢复
- 否决后，恢复请求将被终止

## 安全设置

### 生物识别

- 支持指纹识别和面容识别
- 要求 Class 3（强）生物识别
- 后台 30 秒自动锁定会话

### 防截屏保护

以下屏幕启用防截屏保护：

- 助记词显示页面
- Vault 解锁页面
- 设备详情页面

### 设备完整性验证

Aeternum 使用 Play Integrity API 验证设备完整性：

- 验证通过：正常使用所有功能
- 验证失败：进入降级模式，仅可查看数据

## 常见问题

### Q: 忘记了助记词怎么办？

**A**: 遗憾的是，如果您忘记了助记词且没有其他已连接的设备，您将无法恢复账户。这是为了保证安全性，我们无法帮您找回助记词。

### Q: 为什么恢复需要 48 小时？

**A**: 48 小时否决窗口是为了防止攻击者在获取您的助记词后立即恢复账户。在此期间，您和其他已连接的设备可以否决恢复请求。

### Q: 如何添加新设备？

**A**: 在主设备的设备管理页面，点击"添加设备"按钮，然后使用新设备扫描显示的 QR 码即可。

### Q: 设备显示"降级"状态怎么办？

**A**: 降级状态通常表示设备完整性验证失败。请尝试：
1. 检查设备是否被 root 或有自定义 ROM
2. 确保 Google Play 服务正常运行
3. 点击"重新验证"按钮

### Q: 如何撤销设备？

**A**: 在设备详情页面，点击"撤销设备"按钮。请注意：
- 撤销后该设备将无法再访问您的数据
- 撤销操作不可撤销
- 您无法撤销当前正在使用的设备
