## 状态机流程

### Mermaid 状态转换图

```mermaid
stateDiagram-v2
    [*] --> Uninitialized: 首次启动
    Uninitialized --> Onboarding: 点击"开始设置"
    Onboarding --> Initializing: 完成助记词备份
    Initializing --> Active: 设备注册完成

    state Active {
        [*] --> Idle: 初始状态
        Idle --> Decrypting: 生物识别成功
        Decrypting --> Idle: 会话锁定/超时
        Idle --> Rekeying: 收到新 Header
        Rekeying --> Idle: 轮换完成

        note right of Idle
            无密钥在内存
        end note

        note right of Decrypting
            DEK/VK 在 Rust 内存
            Kotlin 仅持句柄
        end note

        note right of Rekeying
            执行 PQRR 协议
            影子写入 → 原子提交
        end note
    }

    Active --> Degraded: Play Integrity 失败
    Degraded --> Active: 完整性恢复
    Degraded --> Revoked: 持续失败

    Active --> Revoked: 设备被撤销
    Revoked --> [*]: 清除数据

    note right of Degraded
        只读模式
        禁用敏感操作
    end note

    note right of Revoked
        终态
        不可逆
    end note
```
